About The Robots

Team Detox has opted to use two ground robots robots that work in tandem to showcase the versatility of our solution

Tri-Track Robot

Features:
-Lynxmotion Tri-Track Chassis

-Lynxmotion Sabertooth 2X12 R/C Motor Controller

-Lynxmotion BotBoarduino Shield-Compatible Robot Controller

-Lynxmotion AL5D Robotic Arm mounted with proprietary gripping device

-Manual control using Xbox controller and planned autonomous control with coordinate directions from the HighEye

DaNI Robot

Features:
-Controlled using National Instrument myRIO for autonoumous path navigation using an A* algorithm. Coordinates are received from the HighEye

-HC-SR04 Ultrasonic Sensor modules for real time collision detection

-Compatible with 3d printed scooper

-myRIO will also be used for seagull detection using LabVIEW software anda  digital signals processing algorithm

Innovation

Here are some of the innovative features that Team Detox implemented across both solutions

HighEye System


An Eye in the Sky system that runs on low resource computer such as a Raspberry Pi and a camera utilising OpenCV and ORB

Gives a system overview as well as tracking of the ground robots, wildlife, and waterline and rubbish

Provides the virtual coordinate system for the ground robots

Scooper

Custom 3D Printed scooper made from Polylactic Acid

Attachable to DaNI Robot chassic

Can pick up rubbish that cannot be picked up by gripper

Capacitive Sensor

RS Pro 30mm Flush Capacitive Sensor attached to inside of gripper

8mm detection range 

Can detect the difference in electric field for metal, wood, plastic and glass
 

Extended Gripper


-3D Printed from Polylactate Acid

-Increases functional width of gripper to allow picking up of wider objects such as cans and bottles

Features

Our solution incorporates multiple microprocessors to manage the workload to increase efficiency.

Deployable

Both robot are easily deployed into the field and can easily be controlled or monitored. Their surroundings are also easily monitored using the eye in the sky functionality as well as the GUI

Full Manual Control Mode

The Tri-Track Robot can be used in tandem with a manual controller to control it's movement as well as the movement of the arm and gripper. 

Semi-Autonomous

Pathfinding and object detection works in an autonomous way however it will require manual intervention to pick up the rubbish once there.

Durable

Both robots have been made from components acquired from reputable companies who specialise in robotics. The chassis has a robust interior/exterior tracked platform. The solution has also been tested to work over 30+ minutes per charge.
